# Colheita Digital Design System

Bem-vindo ao sistema de design da **Colheita Digital** - uma plataforma inovadora de tokenização para pequenos empreendedores.

## 🎯 Sobre o Projeto

A Colheita Digital é uma plataforma universitária que permite pequenos empreendedores tokenizarem seus ativos produtivos e criarem um ecossistema de fidelidade local através da tecnologia blockchain.

## 🏗️ Arquitetura

Este projeto segue os princípios da **Clean Architecture**:

- **Domain Layer**: Entidades e interfaces de repositório
- **Infrastructure Layer**: Implementações de repositório (mock data)
- **Presentation Layer**: Componentes de UI e seções
- **Shared Layer**: Sistema de design e utilitários

## 🎨 Sistema de Design

### Cores Principais

- **Primary**: Verde (#22c55e) - Representa crescimento e sustentabilidade
- **Secondary**: Azul claro - Para elementos de apoio
- **Background**: Branco/Cinza claro - Para legibilidade
- **Foreground**: Cinza escuro - Para texto principal

### Tipografia

- **Headings**: Poppins (600-700) - Para títulos e destaques
- **Body**: Inter (400-500) - Para texto corrido e interface

### Componentes

#### Design System
- **Button**: Botões com múltiplas variantes e tamanhos
- **Card**: Cartões flexíveis para conteúdo
- **Container**: Containers responsivos para layout

#### Layout
- **Header**: Navegação principal com menu responsivo
- **Footer**: Rodapé com links e informações da empresa

#### Sections
- **HeroSection**: Seção principal com call-to-action
- **ProblemSection**: Apresenta os problemas que resolvemos
- **SolutionSection**: Explica como a tokenização funciona
- **BenefitsSection**: Destaca os diferenciais da plataforma
- **CTASection**: Chamada final para ação

## 🚀 Tecnologias

- **Next.js 15**: Framework React com App Router
- **TypeScript**: Tipagem estática
- **Tailwind CSS**: Estilização utilitária
- **Storybook**: Documentação de componentes

## 📱 Responsividade

Todos os componentes são desenvolvidos com abordagem mobile-first e testados em:

- **Mobile**: 375px+
- **Tablet**: 768px+
- **Desktop**: 1024px+
